# CyberSchoolSystem-Server

## Compiling
Flags:
* dev:
    Use development settings (`config/settings-devel.yml`)
    (i.e reload templates on change and add a dummylogin)
* noAuth:
    Do not compile the authentication modules.
    (i.e the api can be used from curl)
